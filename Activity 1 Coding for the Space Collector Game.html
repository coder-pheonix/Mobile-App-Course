Splash Screen Logic

In this lecture we will be creating a logic to make the "Robot" sprite move.

Logic

  • Click on the "Blocks" tab.
  • Click on the "Play" and drag the "when Play click" block.
  • Go to the "Control" block and drag the "navigate to Screen 1" block.
  • Change the "Screen 1" to "Screen 2" in navigate block.
  • Click on the "Screen 2".
  • Go to the "Variables" block and drag the "initialize app variable name to" block.
  • Go to the "Math" and drag the first block.
  • Change the variable "name" to "x".
  • Go to the "Variables" block and drag the "initialize app variable name to" block.
  • Go to the "Math" and drag the first block.
  • Change the variable "name" to "y".
  • Go to the "Variables" block and drag the "initialize app variable name to" block.
  • Go to the "Math" and drag the first block.
  • Change the variable "name" to "score".
  • Click on the "Screen2" and drag the "when Screen2 opens" block.
  • Go to the "Variables" block and drag the "set app variable x" to block.
  • Go to the "Math" and drag the first block.
  • Change the value from "0" to "40".
  • Go to the "Variables" block and drag the "set app variable x" to block.
  • Select "app variable x" from the drop list.
  • Go to the "Math" and drag the first block.
  • Change the value from "0" to "45".
  • Go to the "Variables" block and drag the "set app variable x" to block.
  • Select the "app variable score" from the drop list.
  • Go to the "Math" and drag the first block.
