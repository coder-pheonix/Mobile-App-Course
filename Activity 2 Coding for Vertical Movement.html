Now, we are going to make the robot move vertically.

  1. Go to the "Up" and drag a the "when Up Click" block.
  2. Go to the "Variables" block and drag the "set app variable x" block.
  3. Go to the "Math" and drag the third block.
  4. Go to the "Variables" block and drag an "app variable x" block.
  5. Change the value from "1" to "10".
  6. Change "app variable x" to "app variable y" from the drop list.
  7. Go to the "Motion" block and drag the "move Sprite8 to x 100 y 100" block.
  8. Change the text from "Sprite8" to "Sprite1".
  9. Go to the "Variables" block and drag an "app variable x" block.
  10. Go to the "Variables" block and drag an "app variable y" block.
  11. Go to the "Down" and drag the "when Down Click" block.
  12. Go to the "Variables" block and drag the "set app variable x" block.
  13. Go to the "Math" and drag the second block.
  14. Go to the "Variables" block and drag an "app variable x" block.
  15. Change "app variable x" to "app variable y" from the drop list.
  16. Change "app variable x" to "app variable y" from the drop list.
  17. Change the value from "1" to "10".
  18. Go to the "Motion" block and drag the "move Sprite8 to x 100 y 100" block.
  19. Change the text from "Sprite8" to "Sprite1".
  20. Go to the "Variables" block and drag an "app variable x" block.
