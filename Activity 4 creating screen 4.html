Creating Score Screen

Quiz Screen and Score Screen

  1. On Screen 4, there is a "Label" and two "Buttons".
  2. Screen 4 shows the points scored in the "Image Quiz".

The steps to create a "Quiz Screen" are as follows.

  1. Click on the "+" to create a new Screen".
  2. Drag the "Label" and place it on Screen 4.
  3. Change the text of the "Label".
  4. Center align the text of the "Label".
  5. Drag the "Button" and place it on Screen 4.
  6. Change the name of the "Button" to "Score2".
  7. Change the change text to "0".
  8. Change the font size from "18" to "40".
  9. Change both "Button" height and width to "120".
  10. Drag another "Button" and place it on Screen 4.
  11. Change the name of the "Button" to "Play Again".
  12. Change the text of the "Button" to "Play Again".
  13. Change the Button height from "38" to "40" and width from "70" to "120".
