Game Screen UI

Now, we will make a "Game Screen" of the "Space Collector Game" app.

This screen consists of the following components:

  1. A canvas component.
  2. Two "Label" components are placed to display the "Score" and "Distance" covered by the sprite.
  3. Four "Button" components, each for each direction arrow key.
