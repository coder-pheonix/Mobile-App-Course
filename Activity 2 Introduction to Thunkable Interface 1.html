Introduction of Thunkable Interface

Thunkable Interface

  1. Now, we will get to know about the interface of Thunkable.
  2. Thunkable consists of two parts:"Design" and "Blocks".
  3. The Design part is visible to the app user and developer both.
  4. The Block part is visible only to the app developer.
