Creating Quiz Screen

Quiz Screen

  1. Screen 3 consists of two components, just like screen 2.
  2. On Screen 3, there is an "Image" and two "Buttons".

The steps to create a "Quiz Screen" are as follows.

  1. Click on the "+" to create a new screen.
  2. Drag the "Button" and place it on Screen 3.
  3. Change the name of the "Button".
  4. Change the text of the "Button".
  5. Change the coordinates, height and width of the "Button".
  6. Drag an "Image" and place it on Screen 3.
  7. Change the coordinates, height and width of the "Image".
  8. Download the image from the link and upload it from your system.
  9. Drag the "Button" and place it on Screen 3.
  10. Change the name of the "Button".
  11. Change the text of the "Button".
  12. Similarly, drag another button. Change the coordinates, height and width of this "Button".
