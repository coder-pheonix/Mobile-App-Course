Adding Web API to "Weather Forecast App"

Screen Weather Forecast

Web API is an application programming interface for the web server. It will provide an interface for websites and client applications to have data access.

The steps to add a Web API are as follows.

  1. Click on the "Blocks".
  2. Click on the "Advanced".
  3. Click the "Web APIs".
  4. Enter "URL", Property and value of Web_API1 block.
  5. Click Web_API1 to change/update API.
